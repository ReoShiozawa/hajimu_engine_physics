# jp-engine_physics — Unity レベル物理エンジン

はじむ言語用の **剛体シミュレーション プラグイン**です。  
Unity の `Rigidbody` / `Collider` / `Joint` / `Physics.Raycast` に相当する機能を Pure C で実装しています。

## 特徴

| カテゴリ | 内容 |
|----------|------|
| **剛体 (Rigidbody)** | Dynamic / Kinematic / Static、質量・減衰・物理マテリアル |
| **コライダー** | 球・箱・カプセル、ローカルオフセット対応 |
| **衝突検出** | AABB ブロードフェーズ + ナロウフェーズ 6 ペア全対応 |
| **衝突応答** | インパルス法、Baumgarte 安定化、Coulomb 摩擦 |
| **慣性テンソル** | 形状別に自動算出（球/箱/カプセル） |
| **ジョイント** | 固定・距離・バネ・ヒンジ（角度制限付き） |
| **レイキャスト** | Ray / SphereCast、OverlapSphere / OverlapBox |
| **レイヤーシステム** | 32 レイヤー × 32 レイヤーの衝突マトリクス |
| **スリープ** | 速度閾値 + 60 フレームカウンタ |
| **サブステップ** | デフォルト 4、精度と速度を調整可能 |

## インストール

### ✅ 推奨: はじむパッケージマネージャー

```bash
hajimu パッケージ 追加 ReoShiozawa/hajimu_engine_physics
```

インストール後すぐに使えます:

```jp
プラグイン取込("engine_physics")
```

### ソースからビルド

```bash
# リポジトリをクローン
git clone https://github.com/ReoShiozawa/hajimu_engine_physics.git
cd hajimu_engine_physics

make          # ビルド → build/engine_physics.hjp
make install  # → ~/.hajimu/plugins/engine_physics/
```

**要件**: はじむ処理系 (`nihongo`) / macOS または Linux / `cmake >= 3.20`

## クイックスタート

```jp
プラグイン取込("engine_physics")

/*  ワールド作成  */
物理ワールド作成()
重力設定(0, -9.81, 0)

/*  地面 (Static 箱コライダー)  */
地面 = 剛体作成()
剛体タイプ設定(地面, 2)        /* 2 = Static */
箱コライダー(地面, 50, 0.5, 50)
位置設定(地面, 0, -0.5, 0)

/*  ボール (Dynamic 球)  */
ボール = 剛体作成()
球コライダー(ボール, 1)
質量設定(ボール, 1)
物理マテリアル設定(ボール, 0.3, 0.7, 0.5)  /* 摩擦, 跳ね返り, スピン */
位置設定(ボール, 0, 10, 0)

/* ─── メインループ ─── */
繰り返し:
    物理更新(0.016)
    位置 = 位置取得(ボール)
    表示("Y = " + 位置[1])
```

## API リファレンス

### ワールド管理

| 関数 | 引数 | 戻り値 | 説明 |
|------|------|--------|------|
| `物理ワールド作成()` | — | 真偽 | ワールドを初期化 |
| `物理ワールド削除()` | — | — | ワールドを解放 |
| `物理更新(dt)` | dt: 秒 | — | シミュレーションを 1 ステップ進める |
| `重力設定(x,y,z)` | — | — | 重力ベクトルを設定（デフォルト: 0,-9.81,0） |
| `サブステップ設定(n)` | n: 整数 | — | 精度向上のためのサブステップ数 |
| `レイヤー衝突設定(a,b,enable)` | — | — | レイヤー間の衝突ON/OFF |

### 剛体

| 関数 | 引数 | 戻り値 | 説明 |
|------|------|--------|------|
| `剛体作成()` | — | ID | 剛体を作成してIDを返す |
| `剛体削除(id)` | — | — | 剛体を削除 |
| `位置設定(id,x,y,z)` | — | — | ワールド座標を設定 |
| `位置取得(id)` | — | [x,y,z] | ワールド座標を取得 |
| `回転設定(id,rx,ry,rz)` | Euler角(度) | — | 回転を設定 |
| `回転取得(id)` | — | [rx,ry,rz] | Euler角(度)を取得 |
| `速度設定(id,x,y,z)` | — | — | 線速度を設定 |
| `速度取得(id)` | — | [x,y,z] | 線速度を取得 |
| `角速度設定(id,x,y,z)` | rad/s | — | 角速度を設定 |
| `角速度取得(id)` | — | [x,y,z] | 角速度を取得 |
| `質量設定(id,m)` | kg | — | 質量を設定 |
| `減衰設定(id,lin,ang)` | 0〜1 | — | 線形・角度減衰 |
| `物理マテリアル設定(id,frict,bounce,spin)` | — | — | 摩擦・跳ね返り・スピン |
| `剛体タイプ設定(id,type)` | 0=Dynamic,1=Kinematic,2=Static | — | |
| `トリガー設定(id,bool)` | — | — | トリガー(通過あり)モード |
| `レイヤー設定(id,layer,mask)` | — | — | レイヤー番号とマスク |
| `位置固定(id,x,y,z)` | 真偽 | — | 各軸の位置ロック |
| `回転固定(id,x,y,z)` | 真偽 | — | 各軸の回転ロック |
| `スリープ設定(id,bool)` | — | — | 強制スリープON/OFF |
| `スリープ中(id)` | — | 真偽 | スリープ状態を確認 |
| `タグ設定(id,tag)` | 整数 | — | ユーザー定義タグ |
| `タグ取得(id)` | — | 整数 | タグを取得 |

### コライダー

| 関数 | 引数 | 説明 |
|------|------|------|
| `球コライダー(id, radius)` | radius: m | 球形コライダー |
| `箱コライダー(id, hx, hy, hz)` | 半径 m | 箱形コライダー |
| `カプセルコライダー(id, radius, height)` | — | カプセル形コライダー |
| `コライダーオフセット(id, x, y, z)` | — | コライダーの中心オフセット |

### 力 / インパルス

| 関数 | 説明 |
|------|------|
| `力追加(id, x, y, z)` | 継続的な力 (F=ma で蓄積) |
| `衝撃追加(id, x, y, z)` | 即座の速度変化 |
| `トルク追加(id, x, y, z)` | 回転力 |
| `点力追加(id, fx,fy,fz, px,py,pz)` | ワールド座標 p に力を加える（トルクも発生） |
| `力消去(id)` | 蓄積された力をリセット |

### ジョイント

| 関数 | 引数 | 戻り値 | 説明 |
|------|------|--------|------|
| `固定ジョイント(a, b)` | ID | JointID | 完全固定 |
| `距離ジョイント(a, b, min, max)` | m | JointID | 距離制限 |
| `バネジョイント(a, b, rest, k, damper)` | m, N/m | JointID | バネ-ダンパー |
| `ヒンジジョイント(a, b, ax,ay,az, minA, maxA)` | 度 | JointID | 1軸ヒンジ |
| `ジョイント削除(id)` | — | — | ジョイントを削除 |

### レイキャスト / クエリ

| 関数 | 戻り値 | 説明 |
|------|--------|------|
| `レイキャスト(ox,oy,oz, dx,dy,dz [,maxD, mask])` | [hit, id, t, nx,ny,nz] | 光線と最初の衝突 |
| `球キャスト(ox,oy,oz, dx,dy,dz, r [,maxD, mask])` | [hit, id, t, nx,ny,nz] | 球スイープ |
| `球重複検出(cx,cy,cz, r [,mask])` | [id, ...] | 球内の全剛体ID |
| `箱重複検出(cx,cy,cz, hx,hy,hz [,mask])` | [id, ...] | 箱内の全剛体ID |
| `衝突確認(a, b)` | 真偽 | 2剛体の衝突判定 |
| `トリガー確認(a, b)` | 真偽 | トリガー重複判定 |

### ユーティリティ

| 関数 | 戻り値 | 説明 |
|------|--------|------|
| `前方向取得(id)` | [x,y,z] | ローカル -Z 軸のワールド方向 |
| `上方向取得(id)` | [x,y,z] | ローカル +Y 軸のワールド方向 |
| `右方向取得(id)` | [x,y,z] | ローカル +X 軸のワールド方向 |
| `速さ取得(id)` | float | 速度の大きさ (m/s) |

| `接触情報取得(i)` | [nx,ny,nz, depth] | i 番目の法線と貫通深さ |

## 応用例

### バネで繋いだ 2 球

```jp
プラグイン取込("engine_physics")
物理ワールド作成()

球A = 剛体作成()
球コライダー(球A, 0.5)
質量設定(球A, 1)
位置設定(球A, 0, 5, 0)

球B = 剛体作成()
球コライダー(球B, 0.5)
質量設定(球B, 1)
位置設定(球B, 2, 5, 0)

/* 自然長2m、k=200 N/m のバネ */
バネジョイント(球A, 球B, 2.0, 200.0, 10.0)

繰り返し:
    物理更新(0.016)
```

### レイキャストで地形を検出

```jp
プラグイン取込("engine_physics")
物理ワールド作成()

/* 地面を Static 箱で作成 */
地面 = 剛体作成()
剛体タイプ設定(地面, 2)
箱コライダー(地面, 100, 0.5, 100)
位置設定(地面, 0, 0, 0)

/* 上から下へレイを打つ */
結果 = レイキャスト(0, 100, 0,  0, -1, 0,  200)
もし 結果[0]:
    表示("地面まで " + 結果[2] + " m")
```
